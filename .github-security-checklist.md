## âš ï¸ What to Do If You Accidentally Commit Secrets

If you accidentally commit sensitive data:

1. **Don't just remove it in a new commit** - it's still in git history!

2. **Use BFG Repo Cleaner or git filter-branch**:
   ```bash
   # Install BFG
   # Download from: https://rtyley.github.io/bfg-repo-cleaner/
   
   # Remove sensitive file from all history
   java -jar bfg.jar --delete-files .env
   git reflog expire --expire=now --all
   git gc --prune=now --aggressive
   
   # Force push (WARNING: Rewrites history)
   git push --force
   ```

3. **Rotate all exposed secrets immediately**:
   - Regenerate API keys
   - Change passwords
   - Revoke tokens

4. **Inform your team** if it's a shared repository
